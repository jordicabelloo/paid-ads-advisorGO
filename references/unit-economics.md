# Unit Economics for Paid Ads

## The Fundamental Equation

```
Net Profit = Revenue per Customer - COGS - CAC
```

Where:
- **Revenue per Customer** = Price (or LTV for subscriptions)
- **COGS** = Cost of goods/service delivery
- **CAC** = Customer Acquisition Cost (ad spend / customers acquired)

## Ecommerce Unit Economics

### Key Metrics
| Metric | Formula | Target |
|--------|---------|--------|
| AOV | Total Revenue / Orders | >$100 |
| COGS | Product + Shipping cost | <40% of AOV |
| Gross Margin | (AOV - COGS) / AOV | >60% |
| CAC | Ad Spend / Customers | < Gross Margin |
| ROAS | Revenue / Ad Spend | >3:1 |
| Net Profit/Order | AOV - COGS - CAC | >$15 |

### Example: Dropshipping Product
| Line Item | Amount |
|-----------|--------|
| Selling Price | $100 |
| Product Cost | $25 |
| Shipping | $15 |
| **COGS** | **$40** |
| **Gross Margin** | **$60 (60%)** |
| Ad Spend per Customer | $40 |
| **Net Profit per Order** | **$20** |
| **ROAS** | **2.5:1** |

### How Metrics Improve Over Time
Your store at launch = the WORST version it will ever be.

Improvement levers (Davie Fogarty):
1. Better ads → lower CAC
2. Better angles → reach new customer segments
3. Better conversion rate → more sales per visit
4. Better offers → higher AOV (bundles, upsells)
5. Lower COGS → negotiate volume with suppliers
6. Email marketing → sales with $0 acquisition cost
7. Referrals → viral coefficient (free customers)
8. SEO → organic traffic over time

### Breakeven ROAS Calculator
```
Breakeven ROAS = 1 / Gross Margin %

Examples:
- 60% margin → Breakeven ROAS = 1.67:1
- 50% margin → Breakeven ROAS = 2:1
- 40% margin → Breakeven ROAS = 2.5:1
- 30% margin → Breakeven ROAS = 3.33:1
```

Any ROAS above breakeven = profit. Below = losing money.

## Services/Coaching Unit Economics

### Key Metrics
| Metric | Formula | Target |
|--------|---------|--------|
| CPC | Cost per Click | ~$1 |
| CR (Landing Page) | Leads / Clicks | 5-10% |
| CPL | CPC / CR | $5-20 |
| Booking Rate | Bookings / Leads | 30-50% |
| Show Rate | Shows / Bookings | 70-75% |
| Close Rate | Sales / Shows | 30%+ |
| CPA | Ad Spend / Sales | < LTV/3 |

### Example: Coaching Program ($3,000)
| Funnel Stage | Rate | Cumulative | Cost |
|-------------|------|------------|------|
| 1,000 clicks | $1 CPC | $1,000 spend | $1/click |
| 100 leads | 10% CR | | $10/lead |
| 40 bookings | 40% booking | | $25/booking |
| 30 shows | 75% show rate | | $33/show |
| 9 sales | 30% close | | $111/sale |
| **Revenue** | 9 × $3,000 | **$27,000** | |
| **ROAS** | | **27:1** | |
| **Net Revenue** | $27,000 - $1,000 | **$26,000** | |

## SaaS Unit Economics

### Key Metrics
| Metric | Formula | Target |
|--------|---------|--------|
| CAC | Total acquisition cost | < LTV/3 |
| LTV | ARPU × Avg Lifetime | > 3× CAC |
| Payback Period | CAC / Monthly Revenue | < 12 months |
| Trial-to-Paid | Paid / Trials | 10-25% |
| MRR | Monthly Recurring Revenue | Growing |

### Example: SaaS ($49/mo)
| Line Item | Amount |
|-----------|--------|
| CPC (Google) | $3 |
| Landing page CR | 8% |
| Cost per Trial | $37.50 |
| Trial-to-Paid | 20% |
| **CAC** | **$187.50** |
| Monthly Revenue | $49 |
| Avg Lifetime | 18 months |
| **LTV** | **$882** |
| **LTV:CAC** | **4.7:1** |
| **Payback** | **3.8 months** |

## Budget Planning

### Minimum Testing Budgets
| Business Type | Per Product/Offer | Duration |
|---------------|------------------|----------|
| Ecommerce | $1,000-2,000 | 2-4 weeks |
| Coaching | $500-1,000 | 2-3 weeks |
| SaaS | $1,000-2,000 | 3-4 weeks |
| Local Services | $500 | 2-3 weeks |

### When to Increase Budget
- Positive ROAS for 7+ consecutive days
- Statistical significance (enough data points)
- Increase 20-30% at a time
- Wait 3-5 days between increases

### When to Cut Losses
After spending minimum testing budget:
1. Checkout/tracking works ✓
2. Ads are quality (external feedback) ✓
3. Website looks trustworthy ✓
4. Tested 3+ angles/audiences ✓
5. Unit economics don't close ✗
→ **Pivot product/offer, not platform**
